BEHIND WALL SETUP:
┌─────────────────────────────────┐
│ 120V AC Outlet (existing)      │
│         ↓                       │
│ 12V DC Wall Adapter             │
│         ↓                       │
│ Arduino Nano ← 12V DC           │
│         ↓                       │
│ NOYITO Modules                  │
│         ↓                       │

└─────────────────────────────────┘

12V Wall Adapter Output:
[+12V] ──┬── Arduino VIN pin
         └── Cloudline A6 +12V input

[GND]  ──┬── Arduino GND pin  
         ├── Cloudline A6 GND
         └── NOYITO modules ground reference
